# Cursor — презентация для команды

MD-слайды (Marp) на ~15 минут: как пользоваться Cursor в режиме IDE.

## Файлы

- [`presentation.md`](./presentation.md) — слайды + заметки спикера в HTML-комментариях
- [`video-script.md`](./video-script.md) — дословный сценарий на ~15 минут для записи с **живым Cursor** (таймкоды, речь, действия на экране; без слайдов)

## Как показать

1. Расширение **Marp for VS Code / Cursor**, либо [Marp CLI](https://github.com/marp-team/marp-cli)
2. Открыть `presentation.md` → Preview
3. Демо вести в **IDE** (редактор + чат агента), не в Agents Window

Экспорт в PDF/HTML (если установлен Marp CLI):

```bash
npx @marp-team/marp-cli presentation.md --pdf
npx @marp-team/marp-cli presentation.md --html
```
